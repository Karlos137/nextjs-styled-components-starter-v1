# Next.js & styled-components starter template

A starter for Next.js and styled-components.

Keywords:

- JavaScript
- React
- Next.js
- Styled Components
- ESLint
- Prettier
- atomic design

### `Before launch`

```
yarn develop
```

### `develop`

```
yarn develop
```

### `build`

```
yarn build
```

### `start`

```
yarn start
```
