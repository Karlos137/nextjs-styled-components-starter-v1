# Next.js & styled-components starter template

A starter for Next.js (v12) and styled-components (v5).

Keywords:

- JavaScript
- React
- Next.js
- Styled Components
- ESLint
- Prettier

### `Before launch`

```
yarn install
```

### `develop`

```
yarn develop
```

### `build`

```
yarn build
```

### `start`

```
yarn start
```
